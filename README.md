# ❤️ Heart Disease Predictor System Using Machine Learning

## 📌 Project Overview

This project demonstrates how to build a **Machine Learning-based Heart Disease Prediction System** using the **Logistic Regression** algorithm.

Using the **`heart-disease-predictor.csv`** dataset, the notebook walks through the complete machine learning workflow, including data exploration, preprocessing, model training, evaluation, and prediction. The goal is to predict whether a patient is likely to have heart disease based on various medical attributes.

This project provides a practical introduction to **binary classification** and demonstrates how machine learning can assist in healthcare decision-making.

---

## 🎯 Objectives

* Understand the heart disease prediction problem
* Explore and preprocess medical data
* Train a Logistic Regression classifier
* Evaluate model performance using classification metrics
* Build a prediction system for new patient data
* Understand the complete healthcare ML workflow

---

## 📂 Dataset

**Dataset Used:** `heart-disease-predictor.csv`

The dataset contains several medical features that are commonly used to diagnose heart disease, including:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression
* Slope of ST Segment
* Number of Major Vessels
* Thalassemia

**Target Variable**

* **0 → No Heart Disease**
* **1 → Heart Disease**
